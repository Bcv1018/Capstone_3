# Capstone_3
---
Capstone Overview
---
This project analyzes Emporium sales performances through four years(2022-2025). This report is made to analyze the East Region which consists of the states of New York and Connecticut.

This Report was created to help make business decisions based on sales trend, product performances, state contribution, and highest performing books.

---
Pages in the Report

Sales Overview:
* Line Chart- Showing Sales Performance through the years(2022-2025)
* Stacked Bar Chart- Shows Sales Performance by Product Category
* Pie Chart- Shows state contribution in the East Region
* Slicers to filter based on state and year for further analysis
Top Books:
* List of our books and how much they made throughout the years(2022-2025)
* Slicers to filter based on state and year for further analysis
